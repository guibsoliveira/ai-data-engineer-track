# Ciclo 6 — Agentes, MCP e Vitrine Pública — contrato

**Janela estimada:** dez 2026 - jan 2027 · **Bancada:** projeto-vitrine
**Status:** aguardando (detalhamento no checkpoint do Ciclo 5)

## Objetivo

Cruzar a fronteira de consumidor para construtor: você USA MCP todos os dias —
aqui você constrói um servidor MCP e um agente com tool use, e publica a vitrine
completa. Ao final deste ciclo, o seu GitHub responde sozinho a pergunta "o que
esse cara sabe fazer?".

## Escopo previsto

1. Anatomia de um agente: loop, tool use, modos de falha, guardrails.
2. Construir um servidor MCP simples expondo os dados da vitrine como ferramentas.
3. Observabilidade: tracing de chamadas, custo e latência (LLMOps básico).
4. Vitrine v1 pública: ingestão → dbt → BigQuery → RAG com evals → agente MCP,
   com README de decisões (o formato que recrutador internacional lê).
5. Revisão geral da trilha: simulado de entrevista técnica cobrindo os 6 ciclos.

## Depois deste ciclo

Sprint de mercado (fev-jun 2027): aplicações BR premium + internacionais, mock
interviews em inglês (a trilha de inglês já terá chegado na fase STAR), leitura
de vagas vira rotina de candidatura.
